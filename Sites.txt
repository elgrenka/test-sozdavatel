Добавление доменов в настройках веб-сервера:

Убедитесь, что ваш веб-сервер (например, Apache или Nginx) настроен для обслуживания нескольких
доменов.
Добавьте виртуальные хосты для каждого домена, указав соответствующую корневую директорию для
каждого сайта.
Настройка многосайтовости в 1C-Bitrix:

Откройте административную панель 1C-Bitrix.
В разделе "Настройки" -> "Настройки продукта" выберите вкладку "Сайты".
Добавьте два новых сайта с соответствующими доменами, указав корневую директорию для каждого
сайта.
Для каждого сайта установите необходимые параметры, такие как язык, кодировку, шаблон и т.д.
Создание и настройка шаблонов:

Создайте два шаблона для сайтов в папке /bitrix/templates/.
Назовите шаблоны соответствующим образом, например, template_site1 и template_site2.
В каждом шаблоне определите необходимую структуру, стили и компоненты для вывода информации на
сайте.
Назначение шаблонов для сайтов:

В разделе "Настройки" -> "Настройки продукта" выберите вкладку "Сайты".
Для каждого сайта выберите соответствующий шаблон, указав его название (например, template_site1
для первого сайта и template_site2 для второго сайта).
Создание общего инфоблока "Новости":

В разделе "Инфоблоки" создайте новый инфоблок "Новости" с необходимыми свойствами, такими как
заголовок и текст новости.
Добавьте несколько новостей в созданный инфоблок.
Вывод списка новостей на главной странице каждого сайта:

В соответствующем шаблоне каждого сайта откройте файл index.php.
Используйте компонент bitrix:news.list для вывода списка новостей из общего инфоблока "Новости".
Настройте параметрыкомпонента bitrix:news.list для выборки и отображения списка новостей на
главной странице каждого сайта:

<?php
$APPLICATION->IncludeComponent(
    "bitrix:news.list",
    "news_list",
    array(
        "IBLOCK_TYPE" => "news",
        "IBLOCK_ID" => 1, // ID общего инфоблока "Новости"
        "NEWS_COUNT" => 5, // Количество новостей для вывода
        "SORT_BY1" => "ACTIVE_FROM",
        "SORT_ORDER1" => "DESC",
        "CACHE_TYPE" => "A",
        "CACHE_TIME" => 3600,
        "SET_TITLE" => "N",
        "PROPERTY_CODE" => array("TITLE", "TEXT"), // Коды свойств для отображения заголовка и текста новости
        "DISPLAY_TOP_PAGER" => "N",
        "DISPLAY_BOTTOM_PAGER" => "N",
    )
);
?>

Сохраните изменения.
